# 日報管理アプリ（Power Apps）

## 概要
SharePointをデータソースとした日報管理アプリです。  
PCでは一覧＋編集の2カラム表示、モバイルでは画面切替を行うレスポンシブ対応を実装しています。  
また、Power Automateと連携し承認フローを組み込んでいます。

---

## 主な機能

### ✅ 日報管理
- 一覧表示（ギャラリー）
- 新規登録
- 編集・更新
- ステータス管理（未対応 / 承認済）

### ✅ モバイル対応
- PC：一覧＋フォームの2カラム表示
- モバイル：一覧 ⇄ 編集画面を切替表示

---

## 承認フロー（Power Automate）

日報登録時に承認フローを自動で実行します。

### 処理フロー
1. ユーザーが日報を登録
2. Power Automateがトリガー
3. 上長に承認依頼を送信
4. 承認／却下結果をSharePointに反映

### 使用技術
- Power Automate
- 承認アクション
- 条件分岐

## SharePoint リスト構成

リスト名：日報

| 列名 | 種類 | 備考 |
|---|---|---|
| タイトル | 1行テキスト | 件名 |
| 内容 | 複数行テキスト | 日報内容 |
| ステータス | 選択肢 | 未対応 / 承認済 |
| 登録者 | ユーザー | 作成者 |
| 作成日 | 日付 | 自動 |

## インポート方法

### Power Apps
1. Power Apps ポータルを開く  
2. 「アプリ」→「パッケージのインポート」  
3. `app` フォルダ内のファイルを指定  

### Power Automate
1. Power Automate を開く  
2. 「マイフロー」→「インポート」  
3. `flow` フォルダ内のファイルを指定  

※接続情報（SharePointなど）は再設定が必要です
